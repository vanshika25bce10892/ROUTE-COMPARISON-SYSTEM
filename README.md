# ROUTE-COMPARISON-SYSTEM
This project focuses on comparing different routes between same initial point and final to understand which route offers the most efficient and comfortable travel experience .  Individuals  find it challenging to determine which route to take . In the absence of an effective system, travellers may squander time, fuel, and financial resources.
Route comparison system

This program helps you to find the shortest and the fastest route by comparing two different routes with same starting point and destination.

Features 
· Multi-route Comparison: Compare unlimited routes side by side
· Cost Analysis: Calculate exact fuel costs for each route
· Time Estimation: Get accurate travel time calculations
· Smart Recommendations: Automatic best route suggestions
· User-friendly Interface: Easy input with validation
· Detailed Reports: Comprehensive comparison metrices.

 programming language: Python
 .Github for project hosting

Usage: 
1. Enter route details when prompted:
   · Route name
   · Distance (in kilometers)
   ·  milage (km/litre)
   · Current fuel price (₹/litre)
   · Average speed (km/hr)
2. View comprehensive comparison result
